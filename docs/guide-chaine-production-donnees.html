Guide de production et integration des donnees

Version de reference pour organiser le travail entre admin plateforme et chargees d'etudes sur SSMSI Dataviz.

A = Thematiques B = Graphiques C = Series D = Observations

1) Vision du projet

Objectif : avoir une chaine robuste, ou la structure reste stable et seules les valeurs sont mises a jour a chaque livraison.

Ce guide formalise qui fait quoi, comment les tables se lient, et comment integrer un bilan annuel provisoire, definitif, une etude ponctuelle ou des enquetes de victimation.
